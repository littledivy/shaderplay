## shaderplay

A WebGPU-based native shader playground.

https://github.com/littledivy/shaderplay/assets/34997667/033830b9-2b9d-4c6f-b362-7292b2daa5e9

Minimum supported Deno version: 2.0.0-rc.5

To run:

```
$ deno run 
     -A
     --unstable-webgpu 
     jsr:@divy/shaderplay 
     examples/waves.wgsl
```
